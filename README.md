# AI Email Reply Generator

## Features
- AI generated email replies
- Multiple tone selection
- Gmail Chrome Extension
- Gemini API Integration
- Spring Boot REST API

## Tech Stack
- Spring Boot
- React + Vite
- Material UI
- Axios
- Gemini API
- Chrome Extension
- Mutation Observer

## Project Structure
backend/
frontend/
extension/

## How to Run

### Backend
mvn spring-boot:run

### Frontend
npm install
npm run dev

### Extension
Load unpacked extension in Chrome
